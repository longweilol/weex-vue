# weex-vue
weex跨平台基础代码

IOS：https://github.com/longweilol/weex-ios-swift

Android：https://github.com/longweilol/weex-android-kotlin

# Run

## install

```
$ npm install
```

## run web

building web pro

```
$ npm run build 
```

building web pro & running service

```
$ npm run dev & npm run serve 
```
